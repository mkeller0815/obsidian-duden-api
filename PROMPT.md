Project: Obsidian Plugin for Duden Mentor API

Objectives:
 - project is tracked in a git repo
 - plugin allows to check spelling of the current text or marked text over the Duden Mentor API
 - changes in the text are highlighted and can be either accepted or declined by the user (or learn a word for later use)
 - in the settings of the plugin the user can enter his API key for personal use 
 - the plugin should be installable via the BRAT plugin

Reasearch:

 - https://docs.obsidian.md/Plugins/Getting+started/Build+a+plugin
 - https://api.duden.io/#introduction
 - https://github.com/TfTHacker/obsidian42-brat
